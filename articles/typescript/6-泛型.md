### 泛型

> 对于一些在调用前不清楚数据类型的值，我们可以选择使用泛型，在调用的时候显示指定该类型.

- 使用泛型:
```
function add<T>(param: T): T {
  return param + param;
}

add<number>(1); // 2
add<string>('a'); // 'aa'
```

- 泛型变量:
```
function loggingIdentity<T>(arg: Array<T>): Array<T> {
    console.log(arg.length);
    return arg;
}
loggingIdentity<number>([1, 2, 3]);
```

- 泛型接口:
```
interface Ia {
  <T>(arg: T): T;
}

function fun<T>(arg: T): T {
  return arg;
}
const fun2: Ia = fun;
```
